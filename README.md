# EduTech Academy

Aplicación móvil desarrollada en Kotlin con Jetpack Compose para explorar cursos online, inscribirse y visualizar el progreso de aprendizaje.

---

## Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Android Studio
- Gemini AI
- Git & GitHub

---

## Pantallas

- **Login / Registro** — autenticación con validación
- **Home** — saludo, accesos rápidos y cursos populares
- **Cursos** — catálogo con filtros por categoría
- **Detalle del Curso** — información completa e inscripción
- **Mis Cursos** — cursos inscritos con barra de progreso
- **Perfil** — datos editables del usuario

---

## Cómo correr el proyecto

1. Clona el repositorio
```bash
git clone https://github.com/Laboratorios-Tecsup/Examen-1-.git
```
2. Abre el proyecto en Android Studio
3. Ejecuta las migraciones si es necesario
4. Corre el proyecto en un emulador o dispositivo físico
